## 状態管理

```typescript
interface PublishState {
  activeTab: 'youtube' | 'tiktok' | 'instagram' | 'schedule';
  pendingVideos: Video[];
  publishedVideos: Video[];
  schedule: ScheduledPost[];
  connectedAccounts: {
    youtube: boolean;
    tiktok: boolean;
    instagram: boolean;
  };
  publishSettings: PublishSettings;
}
```

## API連携

| エンドポイント | 用途 |
|--------------|------|
| GET /api/videos/pending | 公開待ち動画一覧 |
| GET /api/videos/published | 公開済み動画一覧 |
| POST /api/publish/youtube | YouTube投稿 |
| POST /api/publish/tiktok | TikTok投稿 |
| POST /api/publish/instagram | Instagram投稿 |
| GET /api/schedule | スケジュール一覧 |
| PUT /api/schedule/:id | スケジュール変更 |
| GET /api/analytics/best-times | 最適投稿時間 |
