## 状態管理

```typescript
interface ResearchState {
  activeTab: 'competitors' | 'trends' | 'keywords';
  competitors: Competitor[];
  selectedCompetitor: Competitor | null;
  trendingKeywords: TrendKeyword[];
  keywordResults: KeywordResult[];
  savedKeywords: string[];
  isLoading: boolean;
}
```

## API連携

| エンドポイント | 用途 |
|--------------|------|
| GET /api/research/competitors | 登録済み競合一覧 |
| POST /api/research/competitors | 競合追加 |
| GET /api/research/competitors/:id/analyze | 競合詳細分析 |
| GET /api/research/trends | トレンドデータ |
| GET /api/research/keywords | キーワード調査 |
| POST /api/research/keywords/save | キーワード保存 |
