### AI提案ロジック

```
入力:
├── ナレッジ（ターゲット像、インサイト、刺さるメッセージ）
├── 競合調査データ（人気動画、トレンド）
├── キーワード調査データ
└── 過去の動画パフォーマンス

出力:
├── 企画タイトル案
├── 採用理由
├── 参考データ（競合動画、トレンド）
└── 想定キーワード
```

## 状態管理

```typescript
interface PlanningState {
  activeTab: 'calendar' | 'list' | 'ai';
  calendarView: 'month' | 'week';
  currentDate: Date;
  projects: Project[];
  selectedProject: Project | null;
  aiSuggestions: AISuggestion[];
  filters: {
    type: 'all' | 'short' | 'long';
    status: string[];
  };
  isLoading: boolean;
}
```

## API連携

| エンドポイント | 用途 |
|--------------|------|
| GET /api/projects | 企画一覧 |
| POST /api/projects | 企画作成 |
| PUT /api/projects/:id | 企画更新 |
| DELETE /api/projects/:id | 企画削除 |
| POST /api/projects/ai-suggest | AI企画提案 |
