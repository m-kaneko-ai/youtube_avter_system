# 5. 台本・メタデータページ

## 基本情報

| 項目 | 内容 |
|-----|------|
| パス | `/script` |
| 認証 | 必要 |
| ロール | Owner, Team |

## タブ構成

| タブ | 内容 |
|-----|------|
| 台本エディタ | 台本生成・編集 |

| サムネイル | サムネイル生成・選択 |

└── エンディング

## 状態管理

```typescript
interface ScriptState {
  activeTab: 'editor' | 'metadata' | 'thumbnail';
  project: Project;
  scripts: {
    gemini: Script | null;
    claude: Script | null;
    selected: 'gemini' | 'claude' | null;
  };
  metadata: {
    titleOptions: string[];
    selectedTitle: string;
    description: string;
    tags: string[];
    hashtags: string[];
  };
  thumbnails: {
    generated: Thumbnail[];
    selected: Thumbnail | null;
  };
  isGenerating: boolean;
}
```

## API連携

| エンドポイント | 用途 |
|--------------|------|
| POST /api/scripts/generate | 台本生成 |
| PUT /api/scripts/:id | 台本更新 |
| POST /api/metadata/generate | メタデータ生成 |
| POST /api/thumbnails/generate | サムネイル生成 |
| POST /api/videos/:id/submit-approval | 承認依頼 |
