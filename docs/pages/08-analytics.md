# 8. 分析・ナレッジページ

## 基本情報

| 項目 | 内容 |
|-----|------|
| パス | `/analytics` |
| 認証 | 必要 |
| ロール | Owner, Team, Client(Premium+) |

## タブ構成

| タブ | 内容 |
|-----|------|
| パフォーマンス | 動画パフォーマンス分析 |
| ナレッジDB | ナレッジ管理・構築 |
| レポート | 定期レポート |

## 状態管理

```typescript
interface AnalyticsState {
  activeTab: 'performance' | 'knowledge' | 'reports';
  dateRange: DateRange;
  kpis: KPIs;
  videoPerformance: VideoPerformance[];
  knowledges: Knowledge[];
  selectedKnowledge: Knowledge | null;
  chatSession: ChatSession | null;
  reports: Report[];
}
```

## API連携

| エンドポイント | 用途 |
|--------------|------|
| GET /api/analytics/kpis | KPI取得 |
| GET /api/analytics/videos | 動画パフォーマンス |
| GET /api/analytics/video/:id | 動画詳細分析 |
| GET /api/knowledges | ナレッジ一覧 |
| GET /api/knowledges/:id | ナレッジ詳細 |
| POST /api/knowledges/:id/chat | チャットセッション |
| GET /api/reports | レポート一覧 |
| POST /api/reports/generate | レポート生成 |
