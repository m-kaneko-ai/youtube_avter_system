# 2. ダッシュボードページ

## 基本情報

| 項目 | 内容 |
|-----|------|
| パス | `/dashboard` |
| 認証 | 必要 |
| ロール | 全員（表示内容はロール別） |

## タブ構成

| タブ | 内容 |
|-----|------|
| 概要 | KPI、進捗サマリー |
| 今日のタスク | 要対応タスク一覧 |
| 通知 | システム通知、承認依頼 |

### KPIカード（Owner/Team）

| カード | 表示内容 | 比較対象 |
|-------|---------|---------|
| 今月制作数 | 制作した動画数 | 前月比 |
| 公開済み | 公開完了数 | 前月比 |
| 承認待ち | 承認待ち件数 | - |
| 総再生数 | YouTube再生回数 | 前月比 |

### KPIカード（Client）

| カード | 表示内容 |
|-------|---------|
| 制作中 | 現在制作中の動画数 |
| 承認待ち | 確認が必要な動画数 |
| 公開済み | 今月公開された動画数 |
| 納品済み | 納品完了数 |

|-----|------|-------|

## 状態管理

```typescript
interface DashboardState {
  activeTab: 'overview' | 'tasks' | 'notifications';
  kpis: {
    producedCount: number;
    publishedCount: number;
    pendingApproval: number;
    totalViews: number;
  };
  recentVideos: Video[];
  tasks: Task[];
  notifications: Notification[];
  isLoading: boolean;
}
```

## API連携

| エンドポイント | 用途 |
|--------------|------|
| GET /api/dashboard/kpis | KPI取得 |
| GET /api/dashboard/recent-videos | 最近の動画 |
| GET /api/dashboard/tasks | タスク一覧 |
| GET /api/notifications | 通知一覧 |
| PUT /api/notifications/:id/read | 既読更新 |
